HTML Images

An image can be shown using <img> tag alonog with its src attribute

< img src="https://placedog.net/400/281?id=117" alt="This is a dog photo." >

< img src="./images/me.jpg" >

Valid image extensions are the following:

  1. png
  2. jpg
  3. gif
  4. svg
  5. webp
